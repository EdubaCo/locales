# Contributing to edubaco/locales

## License

By contributing you agree your work is submitted under the **MIT License** ([LICENSE](LICENSE)).

## DCO sign-off

Every commit must carry:

```
Signed-off-by: Your Name <you@example.com>
```

Add automatically with `git commit -s`.

## Workflow

1. Fork, create a branch, add/edit JSON files.
2. Run `node validate-keys.mjs` to check key parity with `en/`.
3. Commit with `-s` and open a PR.
